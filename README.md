<h1 align="center">codex</h1>

a system info utility built with Qt6.

## installation

```
git clone https://github.com/theoisdumb/codex.git
cd codex
./install.sh
```

## dependencies

- qt6
- python
- pyside6

## commands used

- lscpu
- lsblk
- ip
- free
